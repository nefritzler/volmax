Design Notes

1
Dual Decks — Always Visible

Both decks are stacked vertically and always visible. No tab switching needed. Deck A (LIVE) on top, Deck B (CUE) below. Bridge tint overlay still applies per-deck when crossfader moves.

2
Mode Pill — Tap to Cycle

Each deck header has a compact pill that cycles Chill → Bounce → Hype → Drop on tap. Color-coded: Chill=#4488ff, Bounce=#44cc66, Hype=#ff8800, Drop=#ff2233. Per-deck state, always accessible.

3
Intensity Knob — Rotary Control

CSS rotary knob with 270° arc. Drag up/down to adjust. Controls global FX output level. Arc track fills from 7-o'clock to current position. Tick indicator shows exact position.

4
Vertical Faders — All Output Controls

Master fader (purple, wider, taller) is leftmost. Zone faders FL/FR/BL/BR all vertical. No horizontal sliders in the output section.

5
Sync All + Zone Sync

Sync All sets all zone faders to master value and marks them synced (purple glow). Each zone also has an individual Sync toggle. Synced zones track master in real time. Mute and Sync buttons are side-by-side at zone bottom.
