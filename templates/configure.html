Design Notes

1
Fixture Discovery

Auto-scans local Wi-Fi subnet on open. Shows live online/offline status. Tap a zone pill to reassign any fixture to a different zone instantly.

2
Floor Grid

4×4 spatial grid representing the room. Drag fixtures onto cells to set physical position. Grid order drives backend effects — chase sequences fire in spatial order, not discovery order. Color-coded by zone for at-a-glance layout review.

3
Grid + Zones in Sync

Reassigning a fixture to a new zone on a grid cell updates the zone card immediately — and vice versa. No two-step reconciliation needed.

4
Inline Zone Rename

Tap a zone name to edit it in place (cursor state shown on Booth card). Each fixture belongs to exactly one zone. Removing a fixture from a zone puts it in the unassigned pool (dashed grid cell).

5
Idle Color

Optional fallback color shown when a zone is not actively performing. The ✕ swatch clears the idle color (zone goes dark when idle). No mode or palette selection on this screen.
